# aws-cognito

### Frontend

Create the env file:

```sh
touch frontend/.env.local
```

Add the variables to the file:

```sh
VITE_API_URL="http://localhost:5000/api/data"
VITE_COGNITO_AUTH_URL="https://<domain>.auth.<region>.amazoncognito.com"
```
